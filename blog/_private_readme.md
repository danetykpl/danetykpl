### Theme
Link: https://adityatelange.github.io/hugo-PaperMod/posts/papermod/papermod-installation/

Installing:
```console
git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1
```

Updating:
```console
cs themes/PaperMod
git pull
```

Configuring (add in `config.toml`):
```console
theme = "PaperMod"
```
